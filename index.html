JavaScript API Savollar va Javoblar

JavaScript API nima?
JavaScript API (Application Programming Interface) - bu JavaScript tilida mavjud bo'lgan vositalar va metodlar to'plami.
Fetch API nima?
Fetch API - bu veb-serverlardan ma'lumot olish uchun ishlatiladigan JavaScript API hisoblanadi.
DOM API nima?
DOM API - bu veb-sahifadagi HTML va XML hujjatlarining tuzilmasini boshqarish uchun ishlatiladigan JavaScript API hisoblanadi.
JSON nima?
JSON (JavaScript Object Notation) - bu ma'lumotlar almashish uchun ishlatiladigan eng oddiy va o'qilishi oson format.
AJAX nima?
AJAX (Asynchronous JavaScript and XML) - bu veb-sahifalarning ma'lumotlarni sinxronizatsiyasiz olishiga imkon beradigan texnologiya.
Event Delegation nima?
Event Delegation - bu DOM elementlarida hodisalarni boshqarishning samarali usuli. Bu yordamida hodisalar ota elementda bajariladi, bu esa ko'plab elementlarga bitta hodisa bilan ishlash imkonini beradi.
CORS nima?
CORS (Cross-Origin Resource Sharing) - bu brauzerda bir domen ichida ishlayotgan veb-saytning boshqa domenlardan resurslarni olishga imkon beradigan mexanizm.
Local Storage nima?
Local Storage - bu brauzerda ma'lumotlarni saqlash uchun ishlatiladigan imkoniyat, bu ma'lumotlar brauzer o'chirilganda saqlanadi.
Session Storage nima?
Session Storage - bu brauzerdagi ma'lumotlarni vaqtinchalik saqlashga imkon beradi. Bu ma'lumotlar faqat brauzer oynasi ochiq bo'lganda mavjud bo'ladi.
Promises nima?
Promises - bu JavaScriptda asinxron operatsiyalarni boshqarish uchun ishlatiladigan obyektlar.
Callback funksiyalari nima?
Callback funksiyalari - bu boshqa funksiyalarga argument sifatida beriladigan funktsiyalar.
Event Loop nima?
Event Loop - bu JavaScriptni asinxron tarzda ishlatishning mexanizmi, bu orqali barcha callback funksiyalar boshqariladi.
Fetch metodining ishlash prinsipi qanday?
Fetch metodi asinxron ma'lumotlarni olish uchun ishlatiladi va u Promise qaytaradi.
Web Storage API nima?
Web Storage API - bu brauzerda ma'lumotlarni saqlashga yordam beradigan API, u LocalStorage va SessionStorage'ni o'z ichiga oladi.
Service Workers nima?
Service Workers - bu brauzerda backgroundda ishlaydigan JavaScript skriptlari, ular offline rejimda ishlashni qo'llab-quvvatlaydi.
Geolocation API nima?
Geolocation API - bu foydalanuvchining geografik joylashuvini olish uchun ishlatiladigan API.
Canvas API nima?
Canvas API - bu HTML5 elementiga asoslangan va rasm va grafikalarni dinamik ravishda yaratish imkonini beradigan API.
WebSockets nima?
WebSockets - bu asinxron ikki tomonlama aloqa protokoli, uni JavaScript orqali ishlatish mumkin.
URLSearchParams nima?
URLSearchParams - bu URLdagi so'rov parametrlari bilan ishlash uchun ishlatiladigan JavaScript interfeysi.
Web Workers nima?
Web Workers - bu JavaScript kodi uchun parallel ishlashni ta'minlaydigan mexanizm.
IndexedDB nima?
IndexedDB - bu brauzerda katta hajmdagi ma'lumotlarni saqlash uchun ishlatiladigan API.
Audio API nima?
Audio API - bu audio fayllarni o'ynatish va tahrirlash uchun JavaScriptda ishlatiladigan API.
Vibration API nima?
Vibration API - bu qurilmaning titrashini boshqarish imkonini beruvchi API.
WebRTC nima?
WebRTC - bu brauzerlarda real vaqtli audio va video muloqotni amalga oshirish imkonini beradigan API.
Clipboard API nima?
Clipboard API - bu foydalanuvchining clipboard (kesilgan yoki ko'chirilgan ma'lumotlar) bilan ishlash imkonini beradigan API.
Notification API nima?
Notification API - bu foydalanuvchilarga brauzerda push-xabarlar yuborish imkonini beradigan API.
Fullscreen API nima?
Fullscreen API - bu veb-sahifalarni to'liq ekran rejimida ko'rsatish imkonini beradigan API.
Battery API nima?
Battery API - bu foydalanuvchining qurilmasining batareya holatini bilish imkonini beradigan API.
Performance API nima?
Performance API - bu veb-sahifalarning ishlash ko'rsatkichlarini o'lchashga yordam beradigan API.
File API nima?
File API - bu brauzerda fayllar bilan ishlash imkonini beruvchi API.
